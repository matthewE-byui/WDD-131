Trailmakers Builds Hub - Site Plan

Purpose

The purpose of this website is to showcase my custom Trailmakers vehicle builds and provide a clear, organized, and interactive way for visitors to explore build details, features, stats, and images.

Audience

The intended audience includes Trailmakers players, engineering-focused gamers, classmates, instructors, and potential collaborators interested in viewing or downloading my builds.

Dynamic Elements

This site will use JavaScript to create a dynamic build gallery using an array of objects stored in ES modules. Filters, sorting tools, a Build of the Day feature, and interactive UI elements will be powered using functions, array methods, conditionals, event listeners, and DOM manipulation.

Logo

Colors & Fonts

  • Dark Background: #0D1117
  • Accent Yellow: #FFC400
  • Light Gray: #E0E0E0

Fonts: Orbitron for headings and Inter for body text.

Content

Home Page

Features a hero section, Build of the Day, About the Creator section, and preview gallery.

Child Page (Build Gallery)

Presents builds in a filterable and sortable grid layout with images, stats, and descriptions.
