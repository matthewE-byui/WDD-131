What is SEO (Search Engine Optimization)?

SEO is the practice of improving a webpage so search engines understand it and rank it higher for relevant searches. It has three main pillars:

  • Technical: fast site, mobile-friendly, correct HTML (meta tags, structured data)
  • On-page: clear headings, useful content, keywords placed naturally, accessible images with alt text
  • Off-page: reputation signals such as backlinks from reputable sites

Quick tips you can apply to any page: use a descriptive title tag, an explanatory meta description, semantic headings (H1/H2), optimize images, and make URLs readable.

Tip: When sorting strings, use localeCompare for correct alphabetical order in most languages.

Quick reference

// numeric sort ascending
arr.sort((a,b)=>a-b)

// numeric sort descending
arr.sort((a,b)=>b-a)

// string sort (case-insensitive)
arr.sort((a,b)=>a.localeCompare(b))

// filter values > n
arr.filter(x => x > n)

// sort objects by key
arr.sort((a,b)=>a.key>b.key?1:-1)

// filter objects by property
arr.filter(o => o.score >= 90)

Common SEO checklist (apply to any webpage)

  • Title tag (unique, ~50–60 chars) and meta description (up to ~155 chars) that describe page content.
  • H1 for main topic, H2/H3 for subsections; include target keywords naturally.
  • Images: compress, use descriptive filenames and alt text.
  • Mobile-friendly — responsive layout and fast loading. Use network and Lighthouse tools to test.
  • Use structured data (JSON-LD) for rich snippets where relevant (articles, products, events).
  • Readable URLs (example: /blog/how-to-sort-arrays-js).
