Part 2: Color Switch Challenge

Type a color (red, yellow, or green) and click "Check Color" to see how switch works.

switch (color) {
  case "red":
    console.log("Stop!");
    break;
  case "yellow":
    console.log("Caution!");
    break;
  case "green":
    console.log("Go!");
    break;
  default:
    console.log("Unknown color");
}

Part 3: Event Practice

Click the button or press a key to trigger events below!

Try pressing any key on your keyboard too!

// Example:
button.addEventListener("click", () => {
  console.log("Button clicked!");
});

document.addEventListener("keydown", (e) => {
  console.log("You pressed: " + e.key);
});
