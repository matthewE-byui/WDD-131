How are forms used to gather information?
Forms let users input info (like login or feedback) using fields like text boxes, checkboxes, and buttons. Data is sent to a server via GET or POST.
Problems to watch for when creating forms?
Missing labels, poor validation, no feedback, unclear instructions, or wrong input types can all cause issues.
How can I style forms with CSS?
Use CSS for spacing, colors, borders, and hover effects. Example: style inputs with padding, border-radius, and color schemes.
Differences between type, id, and name?
type: input kind
id: unique for JS/CSS
name: key sent to the server.
