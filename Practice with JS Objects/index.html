What is CSS Flexbox?
Flexbox is a layout system that allows items to align and distribute space within a container, even when their size is unknown or dynamic.
How can I use Flexbox?
Add display: flex; to a container, then use properties like justify-content and align-items to control layout.
Flexbox vs Grid?
Flexbox is 1D (rows or columns). Grid is 2D (rows and columns). Use Flexbox for simpler layouts, Grid for complex page structures.

What is Chrome Lighthouse?
Lighthouse is a built-in Chrome tool that audits web pages for performance, accessibility, SEO, and best practices.
How can it optimize webpages?
It gives a score and recommendations, like reducing image sizes, improving SEO, or making your site more mobile-friendly.

What are social media meta tags?
Meta tags tell social platforms what title, description, and image to show when your page is shared.
Why are they important?
They control how your content looks on social media, increasing click-throughs and engagement.
Which meta tags should I use?
Use og:title, og:description, og:image, and Twitter card tags like twitter:card.
